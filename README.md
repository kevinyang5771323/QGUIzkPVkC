# 宠物管理系统 java537

## 项目概述

宠物管理系统是一款基于前后端分离架构的宠物相关信息管理平台，旨在为用户提供宠物领养、救助、信息发布等一站式服务。

## 技术栈

- **前端技术：** Vue、ElementUI、ECharts
- **后端技术：** SpringBoot、Mybatis-Plus
- **数据库：** MySQL
- **编程语言：** Java

## 功能模块

### 系统角色

- 用户
- 管理员

#### 用户功能

- **首页：**
  - 展示公告列表
  - 宠物科普
  - 介绍流浪宠物
  - 热门活动
- 注册与登录
- 宠物领养
- 宠物救助
- 丢失宠物查看
- 宠物论坛

#### 管理员功能

- 用户管理
- 申请领养管理
- 评论管理
- 流浪动物救助
- 动物走失管理
- 救助站管理
- 帖子管理
- 捐赠管理
- 公告管理
- 科普文章管理
- 活动管理

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8+ 环境，MySQL 5.7+

## 部署步骤

1. 准备所需环境：Java、MySQL
2. 导入数据库文件
3. 部署后端服务
4. 部署前端服务
5. 启动系统
